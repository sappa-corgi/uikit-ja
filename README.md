# UIkit 日本語ドキュメント（意訳・再構成）

このリポジトリは公式 UIkit ドキュメント（Introduction）を元に、日本語で再構成したオリジナルドキュメントと最小サンプルを収めています。

構成:
- docs/introduction.md — UIkit 入門（概要・特徴）
- docs/installation.md — 導入方法（CDN とローカル、ビルドの概略）
- docs/components/grid.md — グリッド／レイアウト
- docs/components/button.md — ボタン
- docs/components/nav.md — ナビゲーション
- docs/components/modal.md — モーダル
- sample/index.html — CDN を使った動作サンプル

使い方:
1. `sample/index.html` をブラウザで開いて動作を確認してください（インターネット接続が必要）。
2. ドキュメントは Markdown なので、必要に応じて静的サイトジェネレータへ組み込めます。

ライセンスと注意:
- ここにある日本語ドキュメントは公式ページの内容を意訳・再構成したオリジナル文書です。公式の正確なAPIやオプションは必ず公式英語ドキュメントを参照してください。

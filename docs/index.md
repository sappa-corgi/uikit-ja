# UIkit 日本語ドキュメント

ようこそ。これは UIkit の公式ドキュメント（Introduction）を元に日本語で再構成したドキュメント集です。

ナビゲーションから目的のページを選んでください。

- 実際の動作サンプルは `../sample/index.html` をブラウザで開いて確認できます。

このサイトは MkDocs を使って静的サイト化されています。ローカルでビルドするにはプロジェクトルート（この `mkdocs.yml` がある場所）で次を実行してください：

```powershell
python -m pip install -r requirements.txt
mkdocs serve
```

ビルドする場合：

```powershell
mkdocs build
```

# UIkit — 入門

この章は UIkit の基礎的な概要を示します。公式ドキュメントの内容を元に、日本語で要点を整理・再構成しています。

## 概要
UIkit は軽量でモジュール化されたフロントエンド UI フレームワークです。CSS と JavaScript で構成され、共通の UI コンポーネント（グリッド、ボタン、ナビゲーション、モーダルなど）を提供します。コンポーネントは HTML のクラスと属性（`uk-` プレフィックス）で操作します。

## 主な特徴
- モジュール式: 必要なコンポーネントだけを読み込むことができ、プロジェクトを軽量に保てる。
- モダンブラウザ対応: 最新の主要ブラウザで動作するよう設計されている。
- 直感的なクラス体系: `uk-grid`, `uk-button` など、クラス名だけで視覚的にわかりやすい。
- アイコンライブラリが同梱されている（`uikit-icons`）。

## 同梱物（パッケージのイメージ）
- `/css` — コンパイル済みの CSS（例: `uikit.min.css`）
- `/js` — UIkit の JavaScript 本体（例: `uikit.min.js`）とアイコン用スクリプト（例: `uikit-icons.min.js`）

## 開発フロー（概略）
1. まずは CDN かダウンロードしたファイルで試す。
2. 必要に応じて公式のビルド手順でソースからコンパイルし、テーマや変数をカスタマイズする。

# コンポーネント: ページネーション（Pagination）

長いリストやページ分割のためのナビゲーションを提供します。シンプルなページ番号リンクを作れます。

例:

```html
<ul class="uk-pagination uk-flex-center" uk-margin>
  <li><a href="#">1</a></li>
  <li class="uk-active"><span>2</span></li>
  <li><a href="#">3</a></li>
</ul>
```

ポイント:
- `uk-pagination` は `uk-flex` 系と組み合わせて配置を調整できる。

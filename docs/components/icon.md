# コンポーネント: アイコン（Icon）

UIkit のアイコンは `uikit-icons` ライブラリで提供され、`uk-icon` 属性で埋め込んで利用します。

例:

```html
<span uk-icon="icon: star"></span>
<button class="uk-button uk-button-primary"><span uk-icon="icon: check"></span> 保存</button>
```

ポイント:
- `uikit-icons` を読み込む必要がある（`uikit-icons.min.js`）。
- アイコンの名前とサイズを属性で指定できる。
